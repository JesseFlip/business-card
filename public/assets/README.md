# Resume File Instructions

## 📄 **Where to Place Your Resume**

Place your PDF resume file here with the exact filename:

```
jesse-flippen-resume.pdf
```

### Full Path:
```
business-card/public/assets/jesse-flippen-resume.pdf
```

## 📋 **Important Notes**

1. **File Format**: Must be a PDF file (.pdf extension)
2. **Filename**: Must be exactly `jesse-flippen-resume.pdf` (lowercase, with hyphens)
3. **File Size**: Keep under 5MB for best web performance
4. **Optimization**: Consider using a PDF optimizer to reduce file size while maintaining quality

## ✅ **How to Add Your Resume**

### Option 1: Copy/Paste (Windows)
```bash
# From your file explorer:
# 1. Navigate to where your resume PDF is saved
# 2. Copy the file (Ctrl+C)
# 3. Navigate to: business-card/public/assets/
# 4. Paste the file (Ctrl+V)
# 5. Rename it to: jesse-flippen-resume.pdf
```

### Option 2: Command Line
```bash
# Navigate to the project directory
cd business-card/public/assets

# Copy your resume from wherever it is
copy "C:\path\to\your\resume.pdf" "jesse-flippen-resume.pdf"
```
